## `lisk-core sdk:link TARGETSDKFOLDER`

Symlink specific SDK folder during development.

```
USAGE
  $ lisk-core sdk:link TARGETSDKFOLDER

ARGUMENTS
  TARGETSDKFOLDER  The path to the lisk SDK folder

EXAMPLE
  sdk:link /path/to/lisk-sdk/sdk
```
